1. Initialize a repository: $ git init
auto create a .git file that are hidden
file will be untracked (U)


2. Stage changes: $ git add hello.cpp(individual file) / git add .(for all file select)
Untracked(U) --> Stage(A)


3. Commit changes: $ git commit -m "Commit message"
Saves a snapshot of staged changes with a clear, concise msg


4. switch commit: 
$ git checkout commitID (switch to another commit by id)
$ git stash (go to last commit)


5. Check status: $ git status
check history


6. Show changes in working directory: $ git diff


7. View commit history: $ git log(full commit history) / $ git log --oneline(short commit history)


8. List branches: $ git branch
see all branch. current branch will be markdown by *


9. Create a branch: $ git branch branchName (by default has a main branch)


10. Switch branches: $ git checkout branchName


11. Merge branches: $ git merge branchName
first you should work on new branch then If that work perfectly then merge with master/main branch

 
12. Clone repository locally: $ git clone ___url__


13. Push changes to GitHub 


14. Fetch and merge changes from remote: if any change from Remote, then want to update on locally.
$ git fetch origin
$ git merge origin/demo
or 
$ git pull (fetch + merge)



15. Create pull request 
Create new branch 
Delete remote branch 
Fork repository
Fetch changes without merging 
Clone with latest commits only 
View file modification history
