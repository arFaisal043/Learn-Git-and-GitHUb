-- Initialize a repository: $ git init
auto create a .git file that are hidden
file will be untracked (U)


-- Stage changes: $ git add hello.cpp(individual file) / git add .(for all file select)
Untracked(U) --> Stage(A)


-- Commit changes: $ git commit -m "Commit message"
Saves a snapshot of staged changes with a clear, concise msg


-- Check status: $ git status
check history


-- Show changes in working directory: $ git diff


-- View commit history: $ git log(full commit history) / $ git log --oneline(short commit history)


-- List branches: $ git branch
see all branch. current branch will be markdown by *


-- Create a branch: $ git branch branchName (by default has a main branch)


-- Switch branches: $ git checkout branchName
